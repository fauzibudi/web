Projects

Bitcoin Price Forecasting with LSTM

The project leverages a Long Short-Term Memory (LSTM) model to forecast Bitcoin prices, leveraging historical data fetched from the CoinGecko API to predict future price trends.
Movie Recommender System

Developed a content-based movie recommender using cosine similarity on TF-IDF vectorized movie metadata.
Stock Price Forecasting with LSTM

The project focuses on forecasting stock prices using a Long Short-Term Memory (LSTM) neural network model, leveraging historical data retrieved from the Alpha Vantage API to predict future price trends.
Heart Attack Risk Prediction using Machine Learning and Deep Learning

Developed and compared multiple models to predict heart attack risk, achieving up to 74% accuracy with AdaBoost.
Customer Segmentation

Applies clustering techniques to segment customers based on their spending behavior on various product categories. Identified 3 key customer segments to inform targeted marketing strategies.
Sentiment Analysis of Gojek Reviews

Analyzes score distribution (pie chart), monthly trends (line chart), and word frequency (word clouds), providing insights for service improvement.
Multivariable Regression Model to Predict House Prices in Boston

Assessed model accuracy using R² and RMSE metrics (R² score = 0.7447, RMSE = 0.2064).
Customer Churn Prediction for E-commerce Optimization

This project uses Logistic Regression, Decision Tree, Random Forest, SVM, and XGBoost with ROC-AUC to predict churn and enhance retention strategies from e-commerce data.
Optimization of Flight Program Loyalty with RFM-Based Customer Segmentation

Performed K-Means clustering on a flight program dataset using RFM metrics to segment customers into three groups (medium-value, low-value, high-value), offering tailored recommendations to enhance engagement and loyalty.
Dynamic Weather Forecasting with ML and Map Based Interaction

Built a Streamlit app that predicts 7-day weather and air quality using machine learning models, with interactive location input via world map clicks and real-time data from Open-Meteo API.
IndoSentiment: Intelligent Sentiment Analysis for Indonesian Business Reviews

IndoSentiment is an AI-powered application leveraging a fine-tuned IndoBERT model to analyze sentiment in Indonesian customer reviews. Tailored for local businesses, it accurately classifies sentiments as Positive, Negative, or Neutral.
Smart Butterfly Classification: Leveraging AI to Recognize Nature's Beauty

This project develops an intelligent system based on CNN and Transfer Learning for accurate butterfly classification, with the model optimized through tuning to achieve a validation accuracy of 0.86.
Smart Recipe Assistant: Discover and Translate Culinary Recipes with the Power of AI

This project introduces an AI recipe assistant, leveraging Retrieval Augmented Generation (RAG) and advanced language models to robustly comprehend multilingual queries and deliver consistently translated, contextually relevant culinary information.
StoryCraft: Multilingual Narrative Generator

A Gradio-based application that generates engaging, structured stories in Indonesian or English using Groq's llama-4-scout-17b-16e-instruct model.
